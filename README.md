# Costs-Project
Este é um mini sistema desenvolvido como uma aplicação de treinamento para praticar habilidades em React, JavaScript, HTML, CSS e gerenciamento de rotas. O objetivo principal é permitir o gerenciamento de orçamentos de projetos, com funcionalidades para criação de projetos e atribuição de serviços.

### Funcionalidades Principais

- Criação de projetos: Os usuários podem criar novos projetos, fornecendo informações como nome, descrição e detalhes relevantes.
- Atribuição de serviços: Os projetos podem ser vinculados a diferentes serviços, permitindo que os usuários atribuam e gerenciem os serviços relacionados a cada projeto.
- Visualização de detalhes do projeto: Os usuários podem acessar os detalhes de cada projeto, incluindo informações sobre os serviços atribuídos e os custos relacionados.
- Gerenciamento de orçamentos: O sistema permite que os usuários gerenciem os orçamentos de cada projeto, acompanhando os custos, as despesas e as estimativas financeiras.

### Tecnologias Utilizadas

- React: Utilizado para construir a interface do usuário e criar componentes reutilizáveis.
- JavaScript: Linguagem de programação utilizada para a lógica do aplicativo.
- HTML: Utilizado para estruturar e criar os elementos da página.
- CSS: Utilizado para estilizar a interface do usuário e torná-la agradável visualmente.
- React Router: Biblioteca para gerenciamento de rotas no aplicativo.

### Como Executar o Projeto

1. Clone este repositório.
2. Instale as dependências do projeto utilizando o comando `npm install`.
3. Inicie o servidor de desenvolvimento com o comando `npm start`.
4. Acesse o aplicativo em seu navegador, através da URL `http://localhost:3000`.
